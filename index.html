Precision Carton Counter

2022

In an egg sorting machine was a problem: sometimes it did not put exactly 10 eggs to a carton. The fault could not be found, so I got an idea that I could build a system to count the eggs in every carton and notify the user if there is a problem. The system is built on a Raspberry Pi, and it uses OpenCV for processing the images.

  • Raspberry Pi
  • Python
  • OpenCV

A Control panel

2021

I wanted to build a control panel for my home where I could use a Pomodoro timer, see the time, the weather, and the latest result of my favourite hockey team Kookoo, etc.. I used Raspberry Pi's touchscreen display to view all of it and it was very useful. My favourite part of the project is that when Kookoo scores a goal, the system starts a light show for the LED lights in my home.

  • Raspberry Pi
  • Python
  • User Interface

Plant watering system

2022

While being away from my home for a couple of weeks, I needed a way to water my plants. There would have been easier ways to do it, but I wanted to try automating it.
